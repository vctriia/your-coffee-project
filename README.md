# Твоя Кава - Сайт кав'ярні

Сучасний адаптивний веб-сайт для вигаданої кав'ярні "Твоя Кава", створений з використанням HTML5, CSS3 та JavaScript.

**Веб-адреса:** https://vctriia.github.io/your-coffee-project/

**GitHub:** https://github.com/vctriia/your-coffee-project

## Особливості проєкту

### Адаптивний дизайн
- Повна підтримка desktop, tablet та mobile пристроїв
- Responsive верстка з використанням CSS Grid та Flexbox
- Бургер-меню для мобільних пристроїв

### Інтерфейс
- Інтерактивне навігаційне меню з анімаціями
- Слайдер популярних напоїв з автопрокруткою
- Слайдер відгуків клієнтів
- Lightbox галерея для перегляду зображень
- Popup-вікно з пропозицією знижки

### Функціонал
- Меню з фільтрацією за категоріями (кава, чай, десерти)
- Калькулятор замовлень з підрахунком вартості
- Галерея з фільтрами (інтер'єр, кава, десерти, події)
- Контактна форма з валідацією
- Таймер зворотного відліку для акцій
- Інтеграція Google Maps
- Віджет курсу валют

### Технології
- **HTML5** - семантична верстка
- **CSS3** - змінні, анімації, Grid, Flexbox
- **JavaScript (Vanilla)** - без фреймворків
- **Font Awesome 7.0.1** - іконки
- **Google Fonts** - шрифти Montserrat та Pacifico

### JavaScript можливості
- Intersection Observer API для анімацій при скролі
- SessionStorage для відстеження popup
- Drag & Swipe підтримка для карусель
- Smooth scroll для якірних посилань
- Автоматичні слайдери з ручним керуванням
- Валідація форм

## Сторінки сайту

- **Головна** (`index.html`) - Hero секція, переваги, популярні напої, акції, відео
- **Про нас** (`about.html`) - Історія, команда, цінності, відгуки
- **Меню** (`menu.html`) - Каталог з фільтрами, калькулятор замовлень, віджет валют
- **Галерея** (`gallery.html`) - Фото з фільтрацією та lightbox
- **Контакти** (`contacts.html`) - Форма зворотного зв'язку, карта, контактна інформація
- **Автор** (`author.html`) - Інформація про розробника та проєкт
